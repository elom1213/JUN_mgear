# JUN_mgear — mGear / Shifter 리깅 학습 노트

> Autodesk Maya용 모듈식 리깅 프레임워크 **mGear(Shifter)** 를 직접 공부하며 정리한 기술 노트 모음입니다.
> 단순 요약이 아니라 **실무 질문 → 공식 문서 조사 → 검증·정리**의 흐름을 그대로 담았습니다.

채용 담당자·헤드헌터 분들께 — 이 저장소는 제가 **새로운 기술 도메인을 어떻게 스스로 학습하고 구조화하는지**를 보여드리기 위한 자료입니다. 아래 안내를 따라 읽으시면 짧은 시간에 핵심을 파악하실 수 있습니다.

---

## 🎯 이 노트가 보여주는 것 (역량 요약)

- **자기주도 학습 능력** — 실무에서 부딪힐 법한 구체적 문제(예: "팔이 여러 개인 크리쳐를 부위별로 리깅하려면?")를 스스로 정의하고, 공식 문서를 추적해 답을 정리했습니다.
- **기술 문서화 능력** — 개념 → 워크플로우 → 컴포넌트 레퍼런스 → 실전 적용 순으로 계층을 나누고, 노트 간 상호 링크(Obsidian)로 지식 그래프를 구성했습니다.
- **CG / 리깅 / 파이프라인 이해도** — IK/FK, 스키닝, 모션캡쳐 베이크(HumanIK 리타게팅), `mgear.core.anim_utils` 같은 Python API까지 다룹니다.
- **문제 정의의 투명성** — 각 노트는 "어떤 질문에서 시작됐는지"를 명시하고, 아직 검증이 필요한 항목은 TODO로 솔직하게 남겨 두었습니다.

> **학습 목표**: 전신 biped 템플릿이 아니라 **IK가 필요한 부위(팔·다리)만** 골라 컨트롤 리깅을 세팅하고, 나아가 **팔이 여러 개 달린 크리쳐 아바타**에 적용하는 것.
